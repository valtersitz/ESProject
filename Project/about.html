    • 1 - How to use the Web Interface:

    1.1 - Main Page
    On the main page of this Web Interface you can observe the last sample of data and download the file with all the datas received.
    -The Time Stamp is the number of seconds since the sensors started (it will reset every 49 days, so please dowload the datafiles before the reset).
    -The First Sensor Value is the Temperature followed by the Red, Green and Blue values.
    -The Set Point stands for a value that triggers the Command value if the temperature is higher. It can be set in the Settings page.
    -The Command value triggers a I/O for an any use necessary
    -The Datafiles are available to download thanks to the Download Data buttons. The size of these files was limited to allow safe download and easy upload in softwares like Matlab. They are .txt files filled with "TimeXXX_DataXXX:XXX:XXX:XXX:X" lines, the "X" being raw sensors values.
    -The Erase Data button allows you to definitively delete the datafiles.

    1.2 - Settings Page
    On the Settings page, enter the value of the SetPoint you want and press Enter. For now only the Temperature Setpoint is available. The new Value you enter is saved in the module and replace the previous one. To erase the Setpoint parameter, please press the Erase SetPoint button.

    2 - Hardware:
    The ESP is a brand-new, very cheap and powerful Wi-Fi module. It may not always be stable: please insure that the following parameters are respected:

    -3.3 V powered.
    -Decoupling capacities as close to the power lines as possible.
    -VCC and EN pins to 3.3V.
    -GND (and GPIO 15 for models later than ESP03) to Ground.
    -For the HotStandBy module, GPIO16 must be connected to RST.
    -For modules with Red LED, it can be removed to save battery life.
    -For modules with external antennas, the capacitor between PCB antenna and external antenna can be removed to increase range.

    3 - Software
    The programs are coded in Lua 5.1 using NodeMCU 0.9.6 SDK and Esplorer IDE.
    -All modules do not have the same memory available. Please check before use and adapt the sample time for downloading and reseting data files.
    -Due to timeout value of ESP_Sensors module please dowload datafiles at least once every 49 days.
